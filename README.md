# gdpr-hth-notes

## 项目简介

本仓库用于归档和发布一系列独立的 HTML 页面。这些页面以静态形式存放，不依赖后端服务，可直接通过浏览器打开或托管于任意静态文件服务器。

仓库主要记录与 GDPR（通用数据保护条例）相关的个人学习笔记、参考资料整理以及一些信息归档页面的 HTML 版本。内容以通用性文档为主，不针对任何特定网站或域名。

## 目录说明

```
gdpr-hth-notes/
├── README.md          # 本文件
├── pages/             # 存放所有 HTML 页面
│   ├── index.html     # 页面索引（可选）
│   └── ...
└── assets/            # 公共资源（CSS、JS、图片等）
    └── ...
```

- `pages/`：所有可独立访问的 HTML 文件均放置于此目录下，文件名应具有描述性。
- `assets/`：存放页面共用的静态资源文件，便于维护与复用。

## 页面归档说明

每个 HTML 页面均为独立完整的文档，内容主要涵盖：

- GDPR 相关条款的笔记与解读
- 数据保护与合规方面的参考资料整理
- 其他与隐私、数据安全相关的信息归档

所有页面均为作者个人学习与整理所得，不构成法律建议或正式合规文档。读者如需应用于实际场景，请参考官方法规或咨询专业法律顾问。
